Primera Práctica: HDL y Lógica Booleana
----------------------------------------
- Primera Parte

Esta primera parte de la práctica busca la conformación del grupo de trabajo para el curso de Arquitectura De Computadores del periodo 2-2024, una vez establecido el Grupo Macaco, se decidió crear una página web mediante los conocimientos del grupo usando HTML y CSS. De igual manera, se trabajo bajo la aplicación de GitHub sugerida para llevar un registro de los cambios realizados en el código fuente durante la realización de la actividad. Cada integrante hizo su respectiva descripción para finalizar la parte uno de la práctica.

- Segunda Parte

Para la realización de la segunda parte, se dispuso a recordar conocimientos básicos de lógica booleana para realizar las compuertas pedidas usando únicamente compuertas de tipo NAND. Se cumplieron con las actividades solicitadas con su respectiva explicación; de igual manera se resolvieron las preguntas adicionales de la práctica.
